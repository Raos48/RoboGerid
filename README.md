Bot GERID e SAGGESTAO
Este projeto é um robô de automação com interface gráfica que integra e agiliza processos entre os sistemas GERID e SAGGESTAO, eliminando tarefas repetitivas e potencializando a eficiência operacional.
Funcionalidades

Interface gráfica amigável desenvolvida com Tkinter
Importação de arquivo de configuração Excel
Automação de tarefas no sistema GERID
Atribuição e revalidação de acessos
Processamento em lote de múltiplos registros
Feedback em tempo real através de um terminal integrado
Suporte a múltiplas instâncias de execução

Requisitos

Python 3.x
Selenium
openpyxl
webdriver_manager
tkinter

Instalação

Clone o repositório:

   git clone https://github.com/seu-usuario/bot-gerid-saggestao.git


Instale as dependências:

   pip install -r requirements.txt

Uso

Execute o script principal:

   python bot_gerid.py


Na interface gráfica:


Importe o arquivo de configuração Excel
Selecione a funcionalidade desejada
Defina o número de instâncias (se necessário)
Clique em "Executar" para iniciar a automação


Acompanhe o progresso no terminal integrado
